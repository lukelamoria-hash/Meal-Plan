# Mediterranean 90-Day Cut App

Upload every file in this folder to the root of the GitHub repository. Vercel will redeploy automatically.

Current foundation:
- Automatic Day X of 90 dashboard based on July 11, 2026
- Weekly calorie and protein targets
- Daily target-weight range
- Today's rotating meal plan
- Daily habit and weight logging saved in the browser
- Weekly grocery list preview
- Offline/installable PWA shell

No build command is required. This is a static HTML/CSS/JavaScript app.
